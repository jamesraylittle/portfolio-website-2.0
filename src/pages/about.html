I have not touched web technogies in a long time, so I wanted to put together a little protfolio website to (1) make sure I still knew my way around web development, (2) to attempt to showcase who I am as a technologist for potiental employment oppturnities. This website is built using Bootstrap, jQuery, and some help from Font Awesome. I wanted to make something super simple, but extendable. So I created a small Single Page Application. It loads an HTML page based on the query string, from the pages directory. I was using the hash mark at the end of the url to determind which page to load, but I found it difficult to include page ancors, like the ones on my resume page, which lead me to use the query string. You can find the source code, pretty or not, at the GitHub repository.
